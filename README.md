# David Cui — Personal Portfolio

A single-page personal website built with HTML, CSS, and vanilla JavaScript. Features an anime-inspired dark theme with particle effects, scroll animations, and responsive design.

## Live

[davidcui23333.github.io](https://davidcui23333.github.io)

## Tech Stack

- **HTML5** — semantic, accessible structure
- **CSS3** — custom properties, flexbox/grid, keyframe animations, responsive breakpoints
- **JavaScript** — Intersection Observer, canvas particle system, scroll effects

## Features

- Floating particle system (canvas-based sparkles)
- Scroll-triggered fade-in animations with staggered reveals
- Speed-line section dividers (manga aesthetic)
- Gradient text and glow effects
- Mobile-responsive with hamburger nav
- Scroll progress bar
- `prefers-reduced-motion` support

## Run Locally

Clone and open `index.html` in any browser — no build step or server needed.

```bash
git clone https://github.com/DavidCui23333/DavidCui23333.github.io.git
cd DavidCui23333.github.io
open index.html
```
